# Code of Conduct

This project follows the Contributor Covenant code of conduct. We are committed to providing a welcoming and harassment-free experience for everyone.

Key points

- Be respectful and considerate.
- No harassment, discrimination, or abusive behavior.
- If you witness or experience unacceptable behavior, report it to the maintainers via GitHub (open a private issue or contact a maintainer directly).

For the full text, see: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
